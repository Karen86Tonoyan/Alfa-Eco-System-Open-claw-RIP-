# Contributing

This repository is public for evaluation and credibility, but it is not a general open-source contribution target.

## What is welcome

- bug reports
- architecture feedback
- documentation fixes
- partner and integration inquiries

## What is not accepted by default

- unsolicited code contributions to the protected core
- pull requests that expand private execution logic
- requests for operational connector internals

## Collaboration model

If you want to work on deeper modules, use this repo as an entry point and start a B2B or partner conversation first.
