# IP and Access Model

This repository is published as a `source-available / B2B-first` proof repository.

## Public

- architecture and control-plane narrative,
- public preview code for filters, routing, guard flow, and demo,
- test suite proving the visible behavior,
- wrappers and stubs that show deployment shape.

## Private

- deeper operational connectors,
- production policy sets,
- execution internals,
- anti-loop and protected enforcement logic,
- commercial deployment modules and partner channels.

## Principle

The public repository is here to build trust and demonstrate that the system works.
It is not here to transfer the full execution core or erase the commercial advantage of the private stack.

## Access

Deeper modules are available only through explicit collaboration, licensing, or partner agreements.
